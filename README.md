# ReaDB-php
PHP client for ReaDB

## Requirements
### 0MQ library
Installation instruction can be found here: http://zeromq.org/bindings:php
### Snappy library
```Shell
git clone https://github.com/kjdev/php-ext-snappy.git
phpize7.0
./configure
make
sudo make install
```
### msgpack library
```Shell
git clone https://github.com/msgpack/msgpack-php.git
phpize7.0
./configure
make
sudo make install
```
